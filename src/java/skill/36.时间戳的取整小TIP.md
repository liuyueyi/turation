# 时间戳的取整小TIP

一个简单的背景，持有ms为单位的时间戳，需要判断两个时间戳是否为同一分钟

<!-- more -->

先将问题简化下，对s进行取整如何做？

```
time / 1000 * 1000;
```

也就是丢掉后面的三位数；那么对分钟取整呢？

```
time / 60000 * 60000;
```

其实思路和对整百的取整一样，唯一让人感到违和的可能就是时间是60进制的形式，导致一眼看上去，有点诡异
