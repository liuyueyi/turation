shell命令获取当前系统时间并格式化的方式记录

```sh
time=$(date "+%Y-%m-%d %H:%M:%S")
echo "${time}"
```

<!--more-->

**说明**

- date后面有一个空格，否则无法识别命令，
- Y显示4位年份，如：2018；y显示2位年份，如：18。
- m表示月份；
- M表示分钟。
- d表示天，而D则表示当前日期， 如：1/18/18(也就是2018.1.18)。
- H表示小时，而h显示月份。
- s显示当前秒钟，单位为毫秒；S显示当前秒钟，单位为秒。
